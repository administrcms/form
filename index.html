Current status

This project is still work-in-progress. It is built for use with the Laravel Framework (>= 5.2).

The forms are implemented as the FormRequests in Laravel - they self-validate when type hinted in the controller method to which they send the data, if validation fails, they return back without executing the method and preserve the entered value and render their errors.

Defining a Form class

You need to define a class that extends the base abstract Form class from this package.

<?php

namespace App\Http\Forms;

use Administr\Form\Form;
use Administr\Form\FormBuilder;

class TeamForm extends Form
{

    /**
     * Define the validation rules for the form.
     *
     * @return array
     */
    public function rules()
    {
        return [
            'name'      => 'required|max:100',
        ];
    }

    /**
     * Define the fields of the form.
     *
     * @param FormBuilder $form
     */
    public function form(FormBuilder $form)
    {
        $form
            ->text('name', 'Name', ['translated' => true])
            ->submit('save', 'Save');
    }
}
  • Defining Radio groups
<?php

namespace App\Http\Forms;

use Administr\Form\Field\RadioGroup;
use Administr\Form\Form;
use Administr\Form\FormBuilder;

class TeamForm extends Form
{
    /**
     * Define the fields of the form.
     *
     * @param FormBuilder $form
     */
    public function form(FormBuilder $form)
    {
        $form
            ->radioGroup('is_visible', 'Is Visible', function(RadioGroup $group) {
                $group
                    ->radio('yes', ['value' => 1])
                    ->radio('no', ['value' => 0]);
            });
    }
}
